## 2. Математичне підґрунтя програми

Більшість математичних операцій не відрізняються від застосованих в попередніх практичних. Єдине нововведення - обчислення коефіцієнту варіацій ($CV$ чи `coef_variation`)



### Коефіцієнт варіації ($CV$ чи `coef_variation`)

Оскільки різні лампи можуть мати різну базову яскравість ($\bar{X}$), порівнювати показник $\sigma$ між ними некоректно. Для відносного оцінювання стабільності розраховується коефіцієнт варіації у відсотках. Чим він менший, тим стабільніший світловий потік.


$$CV = \left( \frac{\sigma}{\bar{X}} \right) \times 100\%$$

## 4. Підхід до аналізу даних

У коді встановлено межі нормального освітлення як `MIN_NORMAL_LUX = 0.33` та `MAX_NORMAL_LUX = 0.67`. Фізично додаток Phyphox зазвичай експортує відносну яскравість (Relative luminance) або невідкалібровані значення сенсора, якщо вони мають такий порядок. Програма автоматично масштабує логіку перевірки під ці норми, класифікуючи рівень комфорту робочої зони за вказаними критеріями.
