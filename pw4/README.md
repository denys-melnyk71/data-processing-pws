## 2. МАТЕМАТИЧНЕ ПІДҐРУНТЯ ПРОГРАМИ

### А. Лінійна інтерполяція

Найпростіший метод, що з'єднує дві відомі сусідні точки $(x_0, y_0)$ та $(x_1, y_1)$ прямою лінією. Значення всередині пропуску для точки $x$ обчислюється за формулою:

$$y = y_0 + \frac{y_1 - y_0}{x_1 - x_0}(x - x_0)$$

### Б. Поліноміальна інтерполяція (2-й порядок)

Будує через найближчі відомі точки навколо пропуску єдиний поліном другого ступеня (параболу):

$$y = ax^2 + bx + c$$

Коефіцієнти $a, b, c$ знаходяться шляхом розв'язання системи лінійних рівнянь, що проходять через межі розриву даних.

### В. Сплайн-інтерполяція (3-й порядок / Кубічний сплайн)

Замість побудови одного складного полінома для всієї ділянки, простір між точками розбивається на сегменти, і для кожного сегмента $[x_i, x_{i+1}]$ будується свій кубічний поліном:

$$S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^2$$

При цьому метод висуває жорсткі вимоги до гладкості: у точках стику окремих шматочків перша та друга похідні мають бути рівними ($S'_i(x) = S'_{i+1}(x)$ та $S''_i(x) = S''_{i+1}(x)$). Це дозволяє кубічному сплайну ідеально повторювати хвилеподібну форму добових коливань енергоспоживання.

### Г. Метрика оцінки якості: MAPE

Середня абсолютна помилка у відсотках ($MAPE$) розраховується за формулою:

$$MAPE = \frac{100\%}{n} \sum_{i=1}^{n} \left| \frac{y_{true, i} - y_{pred, i}}{y_{true, i}} \right|$$

Де $y_{true}$ — реальне значення енергоспоживання, $y_{pred}$ — значення, підставлене інтерполяцією, а $n$ — кількість пропущених годин.

## 4. ЗАСТОСОВАНИЙ ПІДХІД ТА АНАЛІЗ ДАНИХ

У цій роботі застосовано **метод імітаційного моделювання спотворень часового ряду**. Його суть полягає в тому, що ми беремо повністю чистий, валідований історичний еталон даних, а потім примусово руйнуємо його частину, імітуючи системні аварії чи блекаути (пакетні пропуски). Маючи на руках "істинні" приховані дані, ми отримуємо можливість математично точно перевірити кожен алгоритм на помилку.

В основі підходу лежить **статистичний та структурний аналіз часових рядів**, який включає:

1. **Аналіз неперервності (Хронологічний аналіз):** Перетворення випадкового набору записів у дискретну безперервну шкалу часу з кроком $\Delta t = 1 \text{ година}$.
2. **Аналіз викидів та дублікатів:** Агрегація аномальних подвійних записів за допомогою середнього значення.
3. **Аналіз похибки:** Дослідження відхилень відновлених ліній від тренду реального енергоспоживання за допомогою відносного аналізу похибок.
