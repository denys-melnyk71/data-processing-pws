## 2. МАТЕМАТИЧНЕ ПІДҐРУНТЯ ПРОГРАМИ

### А. Імпутація середнім значенням (Mean Imputation)

Усі пошкоджені елементи замінюються єдиним константним значенням — середнім арифметичним усіх наявних (не-NaN) точок вибірки:

$$\bar{y} = \frac{1}{n} \sum_{i=1}^{n} y_i$$

Де $y_i$ — відомі чисті спостереження, а $n$ — їхня загальна кількість.

### Б. Імпутація медіаною (Median Imputation)

Пропуски заповнюються центральним елементом валідованого варіаційного ряду. Якщо кількість відомих точок $n$ непарна, береться центральне значення. Якщо парна — середнє арифметичне двох центральних елементів:

$$\tilde{y} = \begin{cases} 
y_{\left(\frac{n+1}{2}\right)}, & \text{якщо } n \text{ непарне} \\
\frac{1}{2} \left( y_{\left(\frac{n}{2}\right)} + y_{\left(\frac{n}{2} + 1\right)} \right), & \text{якщо } n \text{ парне} 
\end{cases}$$

### В. Метод LOCF (Last Observation Carried Forward / Попереднє значення)

Значення пропуску $y_t$ прирівнюється до останнього відомого хронологічного спостереження, що передувало розриву:

$$y_t = y_{t-1}, \quad \text{якщо } y_t \in \text{NaN}$$

### Г. Метод NOCB (Next Observation Carried Backward / Наступне значення)

Пропуск заповнюється першим відомим значенням, яке з'являється за таймлайном одразу після пошкодженого блока даних:

$$y_t = y_{t+1}, \quad \text{якщо } y_t \in \text{NaN}$$

### Д. Оцінка похибки (MAPE)

Для розрахунку відхилень використано середню абсолютну похибку у відсотках:

$$MAPE = \frac{100\%}{m} \sum_{j=1}^{m} \left| \frac{y_{\text{true}, j} - y_{\text{pred}, j}}{y_{\text{true}, j}} \right|$$

Де $m$ — кількість штучно створених пропусків ($140$ годин у цьому експерименті).
